About

Like many skiers and climbers I look at the daily Mount Washington avalanche advisory. Every now and then I thought it would be cool to see the day to day changes as an animation.

If I had my choice I would have had more time skiing this winter. But since I couldn't be on the mountain I undertook this project as an interesting way to learn current coding techniques. I use Python to gather the daily avalanche forecasts, and then javascript and HTML for these pages. Certainly more elegant and efficient ways to code this are possible.

Further development

I plan to run this for the 2018-9 season (and actually ski more!).

I don't plan to spend time on the aesthetics even though there is much that could be done, and should be done. The HTML is very basic- retro 1996 style- it's not too horrible on a monitor but the formatting falls apart on small screens. Responsive design would be great but at this time I'm not interested in learning those skills. Other projects are calling.

Contributions are welcome- the code is on GitHub at https://github.com/leoherm/avy You will need your own Google Maps API key.

If you're so inclined, code away.
